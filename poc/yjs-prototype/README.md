# yjs-prototype

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev --host 0.0.0.0
```

### Run the server:

```sh
bun run start-server
```

### Type-Check, Compile and Minify for Production

```sh
bun run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
bun lint
```

configure $$IP$$ in server.js, package.json and src/components/CollaborativeEditor.vue to your ip (or localhost)
